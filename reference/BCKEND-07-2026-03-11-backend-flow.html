The exact rule that identifies this as a sales call
IF event.status == "confirmed"
AND event.end_time <= NOW()
AND external_attendee EXISTS in contacts table
AND external_attendee.deal_id IS NOT NULL
Then: This is a prospect call that just ended. Fire the post-call pipeline.
What Relay does NOT have: the call content. It only knows the meeting happened, who attended, how long it ran. The draft and summary are inferred from what came before and after.
Duration signal: 38 minutes on a 30-minute invite = ran 8 minutes over. Positive engagement signal. Relay records this without any input from Arjun.

The query Today view runs every 30 seconds
SELECT — actions joined with deals and contacts
SELECT a.action_id, a.type, a.title, a.urgency_score, d.warmth_score, d.silence_gap_days, d.stage, c.name, c.company, c.role FROM actions a JOIN deals d ON a.deal_id = d.deal_id JOIN contacts c ON a.contact_id = c.contact_id WHERE a.owner_id = 'arjun_kodo' AND a.status = 'pending' AND (a.snooze_until IS NULL OR a.snooze_until < NOW()) ORDER BY a.urgency_score DESC LIMIT 20
